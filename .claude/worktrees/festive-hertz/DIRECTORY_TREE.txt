DATABASE SCHEMA
===============

7 Models (Phase 1):
  1. Workspace    - Top-level container
  2. Folder       - Optional organization
  3. Snippet      - Reusable text blocks
  4. Prompt       - Main prompt entity
  5. PromptVersion - Immutable snapshots
  6. PromptComposition - Join table (Snippet → Version with rank)
  7. TestRun      - Execution logs

Features:
  ✅ Immutable versioning
  ✅ Many-to-many with ordering
  ✅ JSONB config storage
  ✅ Complete test logging
  ✅ Workspace scoping


CORE FUNCTIONS
===============

lib/compiler.ts (6 functions):
  1. compilePrompt(versionId)          - Fetch & compile version
  2. validateBraces(text)              - Check for {{}} integrity
  3. extractVariables(text)            - Find {{variable_name}}
  4. substituteVariables(prompt, vars) - Replace variables
  5. executePreparedPrompt(...)        - Full compile + substitute
  6. getRequiredVariables(versionId)   - List needed variables


API ROUTES
==========

POST /api/execute
  └─ Execute prompt with LLM API
     Input: {promptVersionId, variables}
     Output: {testRunId, output, tokens, cost, latency}
     Features: OpenAI + Anthropic, cost calc, token counting

GET /api/execute
  └─ Fetch test history
     Parameters: promptVersionId, limit

GET/POST /api/prompts
  └─ Fetch prompts & versions

GET/POST /api/snippets
  └─ Manage snippets with validation

GET /api/testruns
  └─ Test history with stats

GETTING STARTED
===============

1. Install Dependencies
   npm install

2. Setup Database
   npm run db:push
   npm run db:seed   # Optional: adds example data

3. Add API Keys
   cp .env.example .env.local
   # Edit .env.local with OPENAI_API_KEY and/or ANTHROPIC_API_KEY

4. Run Development Server
   npm run dev
   # Visit http://localhost:3000

5. Or use Docker
   docker-compose up -d
   # Available at http://localhost:3000


DEPLOYMENT OPTIONS
==================

Option 1: Local Development
  npm install && npm run db:push && npm run dev

Option 2: Docker (Recommended)
  docker-compose up -d

Option 3: Vercel
  vercel (deploy + set env vars)

Option 4: Self-Hosted
  npm run build && npm start (with PM2/systemd)


ENVIRONMENT VARIABLES
====================

DATABASE_URL
  PostgreSQL connection string
  Example: postgresql://user:password@localhost:5432/promptarchitect

OPENAI_API_KEY
  OpenAI API key (for GPT-4, GPT-3.5-turbo)
  Example: sk-... (from platform.openai.com/api-keys)

ANTHROPIC_API_KEY
  Anthropic API key (for Claude models)
  Example: sk-ant-... (from console.anthropic.com/account/keys)

NODE_ENV
  development | production

NEXT_PUBLIC_APP_NAME
  Application name (visible in browser)


QUICK DEMO WALKTHROUGH
======================

1. Create Snippet (5 min)
   - Go to /snippets
   - Click "New Snippet"
   - Name: "Brand Voice"
   - Content: "You are professional and friendly..."
   - Click "Create"

2. Create Prompt (5 min)
   - Go to /prompts
   - Click "New Prompt"
   - Name: "Product Description"
   - Select "Brand Voice" snippet
   - Template: "Write description for {{product_name}}"
   - Click "Create"

3. Test Prompt (3 min)
   - Go to /test
   - Fill variables (product_name, etc.)
   - Click "Execute Prompt"
   - See output, tokens, cost, latency

4. Compare Versions (2 min)
   - Go to /history
   - Select 2 versions
   - See diff viewer
